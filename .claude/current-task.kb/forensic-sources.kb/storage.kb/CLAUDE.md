# storage.kb — maintenance guide

Sources enumerating **files and on-device databases** — `/sdcard` listings, the
DownloadManager and MediaStore providers, and local copies of pulled artifacts.
One file per capture; see `../CLAUDE.md` for the shared schema and conventions.

Belongs here: a file/dir listing or a content-provider query about stored data.
Not here: package metadata (→ `../packages.kb/`). Keep personal data in
`forensics/` only.
