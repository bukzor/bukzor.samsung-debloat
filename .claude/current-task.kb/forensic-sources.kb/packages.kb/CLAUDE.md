# packages.kb — maintenance guide

Sources describing the **installed-package set and each app's install
provenance**: inventory, install times, installer/source, signatures. One file
per capture. See `../CLAUDE.md` for the shared schema and conventions.

Belongs here: a capture that enumerates packages or exposes per-package install
metadata. Not here: runtime activity (→ `../activity.kb/`) or permission grants
(→ `../capabilities.kb/`).
