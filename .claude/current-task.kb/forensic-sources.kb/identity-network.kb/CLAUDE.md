# identity-network.kb — maintenance guide

Sources covering **accounts/sync, network usage & configuration, system
settings, and device identity** — where exfiltration, MITM, and hijacked
defaults show up. One file per capture; see `../CLAUDE.md` for the shared schema
and conventions.

Belongs here: accounts, data usage, connectivity/proxy/DNS config, settings
dumps, device fingerprint. Not here: per-app permission grants
(→ `../capabilities.kb/`).
