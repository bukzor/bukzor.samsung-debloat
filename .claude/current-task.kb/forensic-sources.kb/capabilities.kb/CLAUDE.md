# capabilities.kb — maintenance guide

Sources exposing **permission grants, special-access, and default-role holders**
— the surfaces adware/spyware abuses (overlay, install-unknown, accessibility,
notification access, SMS, device admin, IME, autofill, default roles). One file
per capture; see `../CLAUDE.md` for the shared schema and conventions.

Belongs here: who holds a sensitive capability or role. Not here: runtime
activity (→ `../activity.kb/`) or install metadata (→ `../packages.kb/`).
