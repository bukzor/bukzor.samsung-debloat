# synthesis.kb — maintenance guide

Meta-analyses that **fuse multiple sources/analyses into a verdict** — naming
patient zero, judging the card-fraud channel, assessing settings tampering. See
`../CLAUDE.md` for the shared schema and lifecycle.

Belongs here: an analysis whose input is many other analyses/findings (its
`consumes` lists the key underlying sources). Not here: single-source analyses
(those live in the topical sub-kbs).
