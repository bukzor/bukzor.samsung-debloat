# open-questions.kb — maintenance guide

One file per **currently-open known-unknown** for this investigation. Presence
in this directory means "still open"; there is no status field.

Belongs here: a specific question whose answer would change what we do next,
with a note on *how* to resolve it.

Does NOT belong here: a question already answered (delete it and write the
answer into `../findings.kb/`), the task plan (`../mission.md`), or established
facts (`../findings.kb/`).

When a question is resolved, **remove its file** and capture the answer as a
finding — keep this directory as the live list of what's still unknown.
